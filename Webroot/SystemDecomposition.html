Functional Decomposition

Current System Breakdown

  1. 1.0 Customer purchases equipment
    1. 1.1 Verify customer status
    2. 1.2 Register client (if new)
    3. 1.3 Complete equipment sale form (salesperson/customer section)
    4. 1.4 Scan UPC for each item
    5. 1.5 Calculate total
    6. 1.6 Process payment
      1. 1.6.1 Enter item information into Equipment Sale Form
      2. 1.6.2 Generate Purchase Agreement
      3. Generate card receipt (if applicable)
    7. 1.7 Save equipment sale form
    8. 1.8 Decrement inventory
  2. 2.0 Customer returns defective/unwanted equipment
    1. 2.1 Process returned inventory
    2. 2.2 Query for proof-of-purchase (if no receipt)
    3. 2.3 Verify return eligibility
      1. 2.3.1 Check receipt date
      2. 2.3.2 Check for defects
      3. 2.3.3 Check for damage
    4. 2.4 Complete equipment return form
    5. 2.5 Process Return
    6. 2.6 Issue receipt
    7. 2.7 Flag returned inventory
  3. 3.0 Customer requests equipment service or maintenance
    1. 3.1 Verify customer eligibility for service
      1. 3.1.1 Register client (if new)
    2. 3.2 Record problem with equipment
    3. 3.3 Generate repair cost estimate
    4. 3.4 Generate service entry form
    5. 3.5 Generate service agreemen
    6. 3.6 Assign technician
    7. 3.7 Technician completes service entry form
    8. 3.8 Process Payment
    9. 3.9 Generate repair service detail report
  4. 4. Process returned equipment
    1. 4.1 Check for defects
    2. 4.2 Scan UPC and Equipment ID
    3. 4.3 Return item to inventory (if non-defective)
      1. 4.3.1 Increment inventory for returned item
    4. 4.4 Contact supplier to return item (if defective)
  5. 5. Order new inventory
    1. 5.1 Generate current inventory repor
    2. 5.2 Generate reorder request for low stock items
    3. 5.3 Compare reorder request with projected demand
    4. 5.4 Compare reorder request with remaining sales season
    5. 5.5 Enter prices for each item in purchase order
    6. 5.6 Enter shipping charges in purchase order
    7. 5.7 Calculate tax, subtotal, and total
    8. 5.8 Combine orders to supplier in one order
    9. 5.9 Email purchase orders
      1. 5.9.1 Fax or mail purchase orders (if no email is provided)
  6. 6. Process new inventory
  7. 7. Generate sales report
  8. 8. Generate salesperson sales/commission report
  9. 9. Generate service summary report
  10. 10. Generate technician service report
